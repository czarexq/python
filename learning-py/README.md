# 🐍 python

> *A collection of small Python scripts built while learning the basics.*

One of my first Python projects — a bunch of mini scripts covering logic, math, games and more.

---

## 📂 Scripts included

| File | Description |
|------|-------------|
| `avarage.py` | Calculates the average of given numbers |
| `bank.py` | Simple bank account simulation |
| `bot_number_guess.py` | Number guessing game vs a bot |
| `calc.py` | Basic calculator |
| `coffe_recomender.py` | Recommends a coffee based on your mood/input |
| `password_rating.py` | Rates the strength of a password |
| `pygame_window.py` | Basic Pygame window setup |
| `shelter.py` | Animal shelter management simulation |
| `six_analize.py` | Analyses a set of six numbers |
